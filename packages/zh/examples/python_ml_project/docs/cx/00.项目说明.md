# 00.项目说明.md

## 项目目标

本示例展示 Python 机器学习项目如何使用 `docs/cx` 管理用例、任务和变更。

## 技术约束

- Python 命令使用 `uv` 管理的解释器。
- 训练、数据和模型对象默认使用完整面向对象方式实现。
- 单元测试不是默认交付物；本示例任务明确声明单元测试时，使用 `unittest` 并与 `src` 结构一一对应。
- 默认完成一个任务文档后只编辑一个生产代码文件；需要第二个代码文件时先拆任务。
