## Project Goal

This example shows how a Python machine-learning project uses `docs/cx` for use cases, tasks, and changes.

## Technical Constraints

- Python commands use a `uv` managed interpreter.
- Training, data, and model objects use strict TDD and strict OOP.
- Unit tests use `unittest` and mirror the `src` structure.
