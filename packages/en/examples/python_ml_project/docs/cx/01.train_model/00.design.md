# 00.design.md

## Reusable Code

| Capability | Existing Entrypoint | Usage |
| --- | --- | --- |
| Config defaults | `src/config` | Training scripts read config objects instead of command-line arguments |

## New Common Code Needed

| Capability | Proposed Entrypoint | Reuse Reason |
| --- | --- | --- |
| Training config object | `src/config/training_config.py` | Several training tasks share defaults and validation boundaries |

## Design Decisions

| Decision | Reason | Impact |
| --- | --- | --- |
| Express default behavior with a config object | Avoid script parameter branching | Tests can construct the config object directly |

## Implementation Boundary

- The current example task only covers the training config class.
- The unit-test file mirrors the source path.
